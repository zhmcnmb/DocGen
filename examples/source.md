DocGen 智能文档生成系统

项目背景：随着企业文档需求量的增长，传统手动编写文档的方式效率低下且格式不统一。DocGen 项目旨在利用大语言模型技术，实现自动化文档生成。

目标：
1. 支持多种源文件格式（Word、PDF、Excel）的自动解析
2. 按照用户自定义模板结构生成文档
3. 提供交互式生成流程，用户可实时审阅和修改

范围：本项目为命令行工具，面向技术开发人员，暂不包括 Web 界面。

技术架构：
- 编程语言：Python 3.10+
- LLM 接口：OpenAI 兼容 API
- 文件解析：python-docx、pdfplumber、openpyxl
- 架构模式：三阶段管线（素材提取 → 逐章生成 → 组装输出）

核心组件：
1. 文件解析器 — 支持 docx/pdf/xlsx/md/txt 格式
2. 模板解析器 — 从 Markdown 提取章节结构
3. LLM 交互层 — 素材提取、章节生成、摘要、修改
4. 会话管理器 — 持久化中间产物，支持中断恢复
5. CLI 交互层 — 用户确认、审阅模式选择

功能模块：
- 文件解析模块：统一的多格式文件解析，输出带来源标记的纯文本
- 模板解析模块：从 MD 模板提取章节标题和描述
- 素材提取模块：根据章节描述从源文件中精准提取素材
- 章节生成模块：逐章节生成文档内容，带前序摘要传递
- 交互模块：确认理解、素材确认、审阅模式选择、逐章审阅、全局修改
- 会话模块：会话持久化和中断恢复
